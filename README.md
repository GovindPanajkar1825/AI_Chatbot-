# AI-Powered Chatbot

An intelligent chatbot for customer support or FAQs using NLP and Transformers.

## Features
- Intent-based responses
- Contextual fallback using HuggingFace Transformers
- Web interface using Flask

## Technologies
- Python (NLTK, Transformers)
- Flask
- HTML/CSS

## Run Locally
1. Install dependencies: `pip install -r requirements.txt`
2. Run: `python app.py`
3. Open browser at `http://127.0.0.1:5000/`
